# 批量添加商品脚本

这个脚本用于批量添加 12 个商品到商品列表中，每个商品使用对应的 `product_X.jpg` 图片。

## 使用方法

1. **确保后台管理服务器正在运行**：
   ```bash
   go run ./cmd/admin
   ```
   默认运行在 `http://localhost:8081`

2. **运行批量添加脚本**：
   ```bash
   go run ./cmd/add-products/main.go
   ```

3. **脚本会自动创建 12 个商品**：
   - 商品 1 - 商品 12
   - 每个商品使用对应的 `product_1.jpg` 到 `product_12.jpg` 图片
   - 价格、库存、秒杀库存会自动递增
   - 秒杀时间设置为从现在开始持续 24 小时
